OPLPOPS-Manager

Java desktop application to manage PlayStation 1 and 2 games for use with Open PS2 Loader and POPS Loader

OPLPOPS-Manager source

This is a Java desktop application designed to efficiently manage your PlayStation 1 and 2 game collection. It offers a wide range of features, including:

  • Downloading images
  • Downloading cheats
  • Downloading game information
  • Downloading game saves
  • Downloading game compatibility data

Notably, the application establishes direct communication with your PlayStation 2 console through FTP, enabling seamless file transfer and direct management on the console itself.

Additionally, the application interacts with a backend server to fetch game information and files, ensuring an up-to-date and comprehensive database.


Redump PlayStation Web Scrapper

Python script for scrapping PlayStation game information from the Redump website

Redump PlayStation Scrapper source

This Python script is designed to scrape PlayStation game information from the Redump website. It utilizes the Beautiful Soup module to parse the HTML pages and extract essential details.

The script gathers comprehensive information about a game disc, including:

  • Number of tracks
  • Pre-gap length
  • Sector size
  • PlayStation libCrypt usage
  • MD5
  • SHA

This valuable data is then stored in a local SQLite database for easy access and management.

The extracted information serves a crucial purpose in another application that I've developed. The said application is specifically tailored for managing PlayStation 1 games intended for use with a PSIO device.


PSIO Assist

Python script for managing PlayStation games for use with a PSIO device

PSIO Assist source

This Python script incorporates the 'tinkter' module to create a user-friendly GUI that empowers users to manage their PlayStation games for use with a PSIO.

The script facilitates the following tasks:

  • Merging of multi-bin games into a single bin file
  • Generating CU2 sheets for the game discs
  • Adding game art
  • Validating game names
  • Creating multi-disc files for games that utilize more than one disc

To accomplish these tasks, the script relies on game disc information previously scraped from the Redump website.


PiS2 Cloud Compiler

Python scripts for compiling PlayStation 2 elf files on a networed Raspberry Pi

PiS2 Cloud Compiler source

The open-source PS2SDK can be compiled on a Unix system in under 30 minutes, but it doesn't compile natively on a Windows machine. To compile the PS2SDK on Windows, MinGW is required, and it can only utilize a single core of the processor. Depending on the CPU speed, compiling the PS2SDK on a Windows machine may take 3-4 hours.

I conducted tests for the PS2SDK compile time on various Raspberry Pi boards. A Raspberry Pi model 1 required over 3 hours for compilation, while a newer Raspberry Pi model 3 could complete the process in just 40 minutes.

As a proof-of-concept, I developed basic Python scripts to compile C source code into PlayStation 2 elf files using a Raspberry Pi connected to the local network. The C source code is uploaded to the Raspberry Pi, where it gets compiled into an elf file using the PS2SDK. Finally, the elf file is returned to the sender. This approach allows PlayStation 2 elf files to be compiled on a system without the need to install the PS2SDK.

Tetris Arduino

A basic Tetris clone game for the Arduino platform

Tetris Arduino source

This is an implementation of the Tetris game, designed to run on an Arduino Uno board. The development of this game was a part of my university coursework, where the objective was to create a portable game using an Arduino board.

The game is written in C++, offering an engaging experience with background music and a display that shows the next shape being generated. To accommodate small LCD monitors, a basic low-level graphics library is utilized for rendering.

All the shapes in the game are generated using a simple function that draws a single square block with a black border.

To ensure smooth performance, the device consists of two Arduino Uno boards. One board handles the Tetris game while the other board is exclusively dedicated to playing the background music.
